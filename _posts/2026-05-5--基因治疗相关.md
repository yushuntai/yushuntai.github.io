### 一、定义与核心理念

- **基因治疗（Gene Therapy）定义**：指通过修改或操纵细胞的基因来治疗或预防疾病的一种医学技术。简单理解，它不再像传统药物那样去“缓解症状”，而是从基因层面去“修正病因”。理想情况下，一次治疗可产生长期甚至终身效果。

### 二、发展历史（里程碑）

| 时期        | 事件                                                         | 意义                           |
| ----------- | ------------------------------------------------------------ | ------------------------------ |
| 1970s       | 首次实现外源基因在体外细胞表达                               | 概念验证                       |
| 1990年      | 首例人体基因治疗：治疗ADA-SCID患儿                           | 开启临床试验时代               |
| 1999年      | 杰西·基辛格（Jesse Gelsinger）因腺病毒载体引发致命免疫反应   | 行业首次重大挫折，推动安全规范 |
| 2000-2005   | 法国X-SCID试验中多名儿童发生白血病（因逆转录病毒插入致癌基因） | 揭示插入突变风险               |
| 2012年      | 欧洲批准Glybera（首个西方市场认可的基因治疗）                | 首次上市，但因价格过高退市     |
| 2017年      | 美国FDA批准Luxturna（治疗RPE65突变致盲）                     | 首个体内给药的AAV基因治疗      |
| 2017年      | 批准Kymriah（CAR-T）                                         | 首个体外基因修饰细胞治疗       |
| 2019年-2023 | Zolgensma（脊髓性肌萎缩症）、Hemgenix（血友病B）、Casgevy（CRISPR编辑治疗镰刀贫血） | 进入基因编辑和体内治愈阶段     |

### 三、技术分类（按作用方式与递送系统）

#### 1. 按作用方式
- **基因替代**：补充功能正常的基因（如腺相关病毒AAV递送RPE65）。
- **基因沉默/敲除**：抑制致病基因表达（如反义寡核苷酸、CRISPR敲除PCSK9降血脂）。
- **基因编辑**：精确修复DNA序列（CRISPR-Cas9、碱基编辑、先导编辑）。
- **细胞改造**：取出患者细胞，修饰后回输（CAR-T、造血干细胞基因治疗）。

#### 2. 按给药方式
- **体内基因治疗**：载体直接注射到患者体内（如静脉、眼内、鞘内）。直接把病毒载体（rAAV、腺病毒、慢病毒等）/递送系统，直接注射到人体内，在患者身体内部完成细胞基因改造、表达治疗基因。核心特点为细胞不用取出来，全程在体内原位操作，如眼科rAAV视网膜基因治疗、肝脏AAV代谢病治疗、肌肉AAV肌营养不良治疗。
  - 优点是操作简单，直接注射、局部/全身给药。
  - 缺点是体内免疫排斥、靶向难度大、载体安全性要求极高。

- **体外基因治疗**：采集细胞（常用造血干细胞或T细胞），在实验室修饰后回输。流程为细胞取出，体外改造，再放回去，如CAR-T细胞治疗、体外编辑造血干细胞治疗遗传病、视网膜类器官移植改造细胞。
  - 优点是体外可控性强、编辑效率高、可以筛选阳性细胞。
  - 缺点是流程复杂、成本极高、周期长。

#### 3. 主流载体（递送系统）
- **病毒载体**（目前主流）：
  - **腺相关病毒（AAV）**：低免疫原性、不整合基因组（优势），但装载量小(≤4.7kb)、幼年给药效差。用于眼、肝、神经、肌肉。
  - 野生型AAV会优先整合人19号染色体AAVS1安全位点，定点整合、风险极低。没有人使用野生型AAV做实验，使用的都是重组型AAV（rAAV），而rAAAV删除了自身的rep/cap元件（介导AAAV定点/随机整合的核心蛋白），只保留了**两端ITR反向重复序列**用来包装目的基因，没有自主复制能力、安全性极高，以游离态（episome）存在于细胞核中，不整合基因组。分裂细胞中游离DNA会随着分裂丢失，而类似视网膜细胞等终末分化细胞，不再分裂，这些游离的rAAAV可以长期稳定表达数年甚至十几年，


  - **慢病毒（Lentivirus）**：可整合宿主基因组、装载量大(可达9kb)，用于造血干细胞体外修饰（CAR-T、SCD）。
  - **腺病毒（Adenovirus）**：高免疫原性，一般仅用于疫苗或肿瘤内注射。
- **非病毒载体**：
  - 脂质纳米颗粒（LNP）：COVID-19 mRNA疫苗已验证。用于CRISPR组分递送（如NTLA-2001治疗甲状腺素转运蛋白淀粉样变）。
  - 电穿孔、纳米颗粒、细胞穿膜肽。





### 四、核心技术工具

1. **病毒载体工程**：去除致病基因，保留外壳与包装信号，加入治疗基因表达盒。
2. **基因编辑工具**：
   - **CRISPR-Cas9**：通过sgRNA引导Cas9切断指定DNA，利用同源定向修复或非同源末端连接实现编辑。
   - **碱基编辑器**：不切断DNA双链，直接转换一个碱基（如A→G，C→T）。
   - **先导编辑器**：可进行小片段插入/删除及所有单碱基替换。
3. **启动子与调控元件**：实现组织特异性表达（如肝特异性启动子）、诱导型表达（四环素调控）。

#### 主要疾病方向
- **单基因病**：血友病A/B、β-地中海贫血、Duchenne肌营养不良、Leber先天性黑朦。
- **恶性肿瘤**：CAR-T治疗CD19阳性淋巴瘤/白血病、BCMA多发性骨髓瘤；溶瘤病毒治疗黑色素瘤。
- **传染病**：CCR5敲除（类似柏林病人治疗HIV）、CAR-T针对HIV感染细胞。
- **眼科/神经退行病**：RPE65、无脉络膜症；帕金森病（递送AADC或GDNF）、脊髓性肌萎缩症。

### 六、挑战与风险

1. **免疫反应**：
   - 预存中和抗体（针对AAV常见人群>30%）；
   - 衣壳特异性T细胞可攻击转导细胞，导致疗效消失或肝酶升高。
   - 炎症风暴（高剂量腺病毒）。

2. **插入突变致癌**：
   - 第一代γ-逆转录病毒易整合于原癌基因附近（X-SCID患者发生T细胞白血病）。
   - 慢病毒整合谱更安全但仍存理论风险。
   - 整合性载体可能引起克隆扩增。

3. **脱靶编辑**：CRISPR在非目标位点切割，可能破坏抑癌基因或关键基因。

4. **载体生产与成本**：
   - AAV需要HEK293细胞贴壁培养，产能低、成本高；
   - 质粒DNA、纯化层析、质控环节导致单剂成本数万至百万美元。

5. **长期持续性**：非整合载体（如AAV）在分裂细胞中稀释；部分患者疗效随细胞更新而减弱。

6. **递送局限**：许多载体无法高效穿越血脑屏障或进入特定实体瘤深部。

### 七、伦理与监管

1. **生殖系基因编辑**（如He Jiankui事件）：禁止用于可遗传的胚胎编辑（多数国家法律禁止）。
2. **知情同意**：需向患者充分说明插入突变、延迟性免疫反应等不确定性风险。
3. **可及性与公平性**：数百万美元的高价，医保支付模式（如按疗效付费、分期支付、风险共担协议）。
4. **孤儿药路径**：多数基因治疗针对罕见病，FDA/EMA提供快速通道、优先审评、罕见儿科疾病优先券。
5. **长期随访**：监管要求对接受整合载体患者进行15年定期监测（检测克隆扩增与恶性肿瘤）。

### 八、未来趋势与总结

- **体内基因编辑**：LNP或病毒样颗粒递送CRISPR用于遗传性转甲状腺素蛋白淀粉样变性（NTLA-2001临床数据）、新生儿肝脏代谢病。
- **通用型（异体）CAR-T**：利用基因编辑敲除TCR和CD52，避免移植物抗宿主病，降低成本。
- **表观遗传编辑**：不改变DNA序列，仅调控基因开启或关闭（CRISPRoff技术）。
- **无病毒递送**：DNA纳米线、外泌体、可电离脂质LNP针对性更强。
- **推进大型常见病**：通过基因治疗降低LDL-C（PCSK9敲除）、HIV功能性治愈（CCR5敲除）。

**总结**：基因治疗已从高风险最后的孤注一掷，成为部分遗传病与血液肿瘤的标准治疗。其最大优势在于“一次给药，长期甚至终生治愈”，但高昂成本、递送系统局限、长期风险和制造复杂度仍是瓶颈。未来随基因编辑安全性的提高和生产工艺的突破，有望更广泛覆盖从罕见病到常见慢性病的巨大跨度。




### 九、临床上对于AAV安全性的防护措施

AAV随机整合→插入gene body→内源基因插入失活敲除→致病/致癌，理论风险真实存在。

临床上的解决逻辑就是：
删整合关键基因+精简低风险载体+临床前全基因组整合位点筛查+局部局限给药+长期安全性随访+特异性弱调控元件设计，
把「随机插入编码区」的概率压到临床可接受的极低水平，远低于疾病本身致残致盲的风险，因此获益>>风险。


二、临床上整套规避策略（从载体设计→工艺→筛选→临床前→临床监控全链条）

1. 载体层面：从源头最大限度降低整合概率

（1）彻底删除病毒复制&整合关键元件

野生型AAV本身具有一定的整合天性，但是会优先整合人19号染色体AAVS1安全位点，定点整合、风险极低；所有实验室使用的都是重组AAV（rAAV），rAAV删掉了rep/cap元件（介导AAAV定点/随机整合的核心蛋白），只保留**两端ITR反向重复序列**和启动子+目的基因；即rAAV没有自主复制能力、安全性极高，以游离态（episome）存在于细胞核中，基本不整合基因组。但！依然存在极低概率随机非特异性整合，不是0风险。

（2）选用短片段、精简载体骨架

载体片段越长、冗余序列越多，整合倾向性越高；
眼科AAV全部做最小化骨架：仅保留ITR+特异性视网膜启动子+目的基因+polyA，无多余垃圾序列。

（3）选用视网膜细胞特异性弱/中等强度启动子

不用强泛表达启动子（CMV这类）：
强启动子不仅表达过强有毒性，还会提升基因组插入后的上下游基因异常激活风险；
眼科常用：RPE65、GRK1、VMD2 视网膜组织特异性启动子，精准靶向、副作用小、整合后连锁扰动更低。

（4）避免载体内部含同源重复序列、转座子样序列

这类序列会大幅提升同源重组、基因组异位整合概率，载体构建阶段直接序列筛查剔除。

 

2. 设计层面：主动避开插入后破坏内源基因的结构隐患

为避免插到gene body里敲除内源基因，设计上会做到：

- 载体内部不携带剪切供体/受体剪接位点，减少插入后异常剪接、内源基因断裂；
- 不携带强增强子、强沉默子，就算偶然插入非编码区，也不会远距离扰动上下游大片段基因调控网络；
- 密码子优化时降低序列同源性，减少和人基因组同源互补导致的同源重组整合。

 

3. 临床前研究：硬性整合安全性全面筛查

新药上临床前，必须全套实验证明整合风险可控：

1. 体内基因组整合测序（Linear amplification-mediated PCR，LAM-PCR）
检测动物视网膜组织中AAV整合位点分布，统计：- 是否偏好插入基因编码区(gene body)
- 是否富集原癌基因附近
- 整合拷贝数、整合热点
2. 长期动物毒性试验（猴、大鼠长期眼部给药）
长期随访观察：肿瘤、视网膜萎缩、异常增殖、基因突变相关病理；
3. 全基因组测序WGS评估插入突变负荷；
4. 药监要求：编码区整合占比必须极低，无致癌热点富集才能过审。

 

4. 给药方式：眼科局部给药，风险物理限定在眼部

视网膜基因治疗是玻璃体腔/视网膜下局部注射：

- 病毒不进入全身循环、不入骨髓、不进入生殖细胞
- 只局限在眼球局部视网膜/RPE细胞
就算发生极低概率不良整合：
仅局部细胞问题，不会全身扩散、不会生殖系遗传、不会全身肿瘤风险
这也是眼科AAV比肝脏、肌肉AAV治疗安全性宽容度高很多的关键。

 

5. 临床随访：长期监测基因突变、增殖异常、肿瘤

上市后临床要求长期随访：

- 眼底镜、OCT长期监测细胞异常增生、结节、肿瘤样病变
- 视力、视野、视网膜结构长期跟踪
- 一旦出现局部异常增殖，可及时干预处理

 

6. 对比：为什么大家更担心肝脏体内AAV，眼科AAV争议小？

- 肝脏细胞持续分裂，游离AAV易丢失，反而整合倾向性被动升高；
- 肝脏血流量大、全身暴露，整合后全身致癌风险高；
- 视网膜细胞是高度分化、不再分裂的终末细胞：
游离态AAV在不分裂细胞里可以长期稳定表达数年甚至十几年，这也是眼科首选AAV的核心原因；
分裂细胞中游离DNA会随分裂丢失，眼科完美避开这个问题，但随机整合的隐患依然是药企、药监核心考核点；所以插入敲除内源基因的风险，理论成立、逻辑通顺，但在眼科rAAV里是极低概率稀有事件，且被全流程工艺死死压制。
