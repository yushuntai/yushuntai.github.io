# 一、概述与基本概念
## 1.1 定义

补体（Complement, C）是存在于人和脊椎动物**血清、组织液、细胞膜表面**的一组经活化后具有酶活性的蛋白质，由30余种可溶性蛋白、膜结合蛋白组成，统称**补体系统**。1894年Bordet发现：新鲜血清中存在一种不耐热成分，可辅助抗体溶解细菌，故称"补体"。补体是**固有免疫 + 适应性免疫衔接的核心效应通路**。



## 1.2 基本特点

| 特征         | 说明                                                         |
| ------------ | ------------------------------------------------------------ |
| **合成部位** | 肝脏（90%以上）；巨噬细胞、内皮细胞、淋巴细胞局部少量合成    |
| **化学本质** | 多为糖蛋白，正常血清含量仅次于白蛋白、球蛋白                 |
| **灭活特性** | 56°C加热30min即可失活（补体灭活血清）                        |
| **活化方式** | 级联酶促反应，前体蛋白（酶原）依次切割活化，逐级放大效应     |
| **作用特点** | 无特异性，不区分抗原种类，可被病原体、免疫复合物、多糖等激活 |
| **代谢**     | 占血清总蛋白约10%，代谢速度极快，半衰期平均仅几分钟          |
| **发育**     | 胎儿发育前三个月开始产生，出生时系统已发育完成，出生后3-6个月达成人水平 |



## 1.3 核心作用

*   **直接“溶解”敌人**：在目标细胞（如细菌）表面“打孔”，使其内容物外泄而死亡。
*   **标记“猎物”**：包裹病原体，给吞噬细胞发出强烈的“吃掉我”信号（即调理作用）。
*   **召唤“援军”**：产生信号分子，吸引免疫细胞奔赴感染战场（即趋化作用）。
*   **清理“战场”**：帮助清除体内的免疫复合物和凋亡的细胞，维持内环境稳定。



# 二、补体系统组分分类

补体系统的成分按生物学功能可分为**三大类**：**固有成分、调节蛋白、补体受体**。

## 2.1 固有补体成分（活化级联核心蛋白）

存在于血浆及体液中，直接参与补体激活级联反应的蛋白，是激活通路的核心分子，按功能通路划分，命名规则：C1~C9；裂解片段小片段加a、大片段加b（**C2例外：C2a为大片段**）

| 通路                    | 成分                                                        |
| :---------------------- | :---------------------------------------------------------- |
| **经典通路**            | C1q、C1r、C1s、C2、C4                                       |
| **旁路通路**            | C3、因子B（FB）、因子D（FD）、因子P（备解素）               |
| **凝集素通路**          | MBL（甘露糖结合凝集素）、MASP1/MASP2（MBL相关丝氨酸蛋白酶） |
| **共同终末通路（MAC）** | C3、C5~C9                                                   |

## 2.2 补体调节蛋白（负向调控，防止自身损伤）

调控补体激活强度，防止补体过度活化损伤自身正常细胞，分**血浆可溶性**、**膜结合型**两类。

**血浆可溶性补体调节蛋白：**C1 抑制物、I 因子、H 因子、S 蛋白等
**膜结合型补体调节蛋白：**CD55（DAF）、CD59、CD46 等

## 2.3 补体受体（CR，介导细胞信号与吞噬）

表达于各类细胞膜表面，特异性结合活化补体片段，介导细胞生物学效应，如：CR1、CR2、CR3、CR4、C5aR、C3aR 等，介导调理吞噬、炎症趋化、免疫活化等功能。

# 三、补体三条激活通路（独立启动，共用终末通路）

补体系统主要通过三条独立的通路被激活，但都会生成 **C3转化酶 → C5转化酶**，最终组装MAC裂解靶细胞。补体只有被激活后才具有生理活性，补体激活有三条途径，根据起始分子的不同，分为**经典通路**、**MBL凝集素通路**和**旁路通路**。三条通路最终都会生成**C3转化酶**，该酶可以能将核心蛋白C3裂解为C3a和C3b片段，这一步是整个补体系统的核心和放大器。随后，生成的C3b与C3转化酶结合，形成**C5转化酶**，正式进入共同的末端通路，即**攻膜阶段**。攻膜阶段，C5转化酶将C5裂解为C5a和C5b。其中C5b与其他补体蛋白(C6、C7、C8和C9)结合，形成**膜攻击复合物(Membrane attack complex, MAC)**，MAC是一个插入细胞膜的管状孔洞结构，该结构会破坏细胞膜的完整性，导致水和离子大量涌入，使病原体（如细菌）因渗透压改变而死亡。

| 特征             | 经典通路                 | MBL凝集素通路    | 旁路通路          |
| ---------------- | ------------------------ | ---------------- | ----------------- |
| **激活物**       | IgM/IgG免疫复合物        | 病原体甘露糖     | 病原体脂多糖/多糖 |
| **起始关键分子** | C1q                      | MBL-MASP         | 自发C3水解        |
| **是否依赖抗体** | 是                       | 否               | 否                |
| **出现时机**     | 感染后期                 | 感染早期         | 感染即刻          |
| **C3转化酶**     | C4b2a                    | C4b2a            | C3bBb             |
| **C5转化酶**     | C4b2a3b                  | C4b2a3b          | C3bBb3b           |
| **主要防御对象** | 胞外菌、病毒（抗体介导） | 真菌、革兰阴性菌 | 革兰阴性菌、真菌  |




# 四、精密的调控与失衡的后果

补体系统有严格的管控，机体内存在多种**补体调节蛋白**（如C1抑制物、H因子、I因子等），它们在不同环节抑制补体的过度激活，确保攻击只针对外来入侵者。

一旦这个平衡被打破，就会导致疾病：
*   **补体缺陷**：易引发反复的细菌感染，尤其是**奈瑟菌**感染。
*   **过度激活**：可能导致自身免疫病（如**系统性红斑狼疮**）或组织炎症损伤。
