# 爻之气
 
1. 一看月、二看日、三看动爻
2. 在月看旺相休囚
3. 在日看长生墓绝
4. 看动爻是否来生扶冲克
 
有气的爻是有生命力的，是可以发挥作用的，无气的爻没有作用。
 
 
 
# 旺相休囚
 
- 春季（寅卯月）：木旺，火相，水衰，土金休囚
- 夏季（巳午月）：火旺，土相，木衰，金水休囚
- 秋季（申酉月）：金旺，水相，土衰，火木休囚
- 冬季（亥子月）：水旺，木相，金衰，火土休囚
- 辰戌丑未月：土旺，金相，火衰，水木休囚
 
五行生克是一种最为原始的状态，四季的更替会让生克发生变化。在春季木极旺的时候，金是克不了木的。五行在四季中因旺衰生命力的不同变化，其生克关系不像原始状态那么顺利。
 
例：
 
- 春天起卦，卦主为木，竞争对手若为金，对方想克卦主，力量太弱，克不了。但到秋季，金旺，竞争对手变强，若可速战速决，至春季就稳定。
- 到秋季，卦主会衰弱。若卦主为金，竞争对手若为木，则宜护到秋季。
  
 
补充说明
 
- 月支的五行是旺
- 月支生者为相
- 生月支者为衰
- 克月支或被月支克为休囚
 

# 长生、墓、绝

**金长生在巳，木长生在亥，火长生在寅，水土长生在申。**

记住三合局即可方便理解；申子辰、寅午戌、巳酉丑、亥卯未；其中第1个就是合出来的五行长生之地，第3个就是合出来的五行的墓库，比如申子辰三合水局，第1个为申，水长生在申，第3个是辰，辰为水库；另外多记一个土也长生在申；

十二长生在六爻中只用到三个位置：长生、墓、绝。
在八字中，十二长生都要去看，而在六爻里面只看这三个即可。
例：若某根爻五行属火，若起卦日为寅日，则该爻在日得长生；若日为戌，则该爻入了日的墓。

金长生在巳；木长生在亥，火长生在寅，水长生在申；寅申巳亥四长生。


 
### 墓库
 
**丑为金库，未为木库，戌为火库，辰为水库。**

墓库在六爻预测中用到的概率很高，墓库不参与爻能量的变化。墓即无气的爻，库是暂时被关起来，还有机会会再出来。
例：甲申月乙丑日，某根爻是申金，在月为旺，在日故为库。若起卦是戌午月乙丑日，则申金爻入了丑土的墓。
例如忌神发动，入库之爻不被忌神所克，可以避凶。

### 入墓规则

**只有3种入墓：入日墓、动爻入变爻墓、静爻入动爻墓；**

**解释：** 
1.只能低层次入高层次的墓，层次高低为：静爻＜动爻＜变爻＜日月。所以静爻不能入静爻墓库，动爻不能入动爻墓库，日月不能入爻墓库，因为日月为大。<br>
2.注意动爻只能入本位变爻之墓库，静爻不入变爻之墓库。另外，墓爻本身的状态也很重要，若墓库自身被月破日破，则其也不具备墓库作用。静爻也可入暗动爻之墓。<br>
3. 动爻只入变爻的墓，且变爻只让动爻入墓，变爻不入墓（包括变爻不入日墓和其它墓，因为变爻只作用于本位动爻，动爻也不入日墓和其它墓，其他爻也不能入变爻的墓）；<br>
4. 静爻入动爻、暗动、日墓；<br>
4. 伏神入飞神、动爻、暗动、日墓；<br>
5. 没有月墓。<br>

（2）墓爻逢合则闭。如寅木化未土，原本入未土之墓，但是此时日为午火，闭合了未土之库，此时寅木不入墓。
（3）墓爻逢冲则开。同样如果墓库为动爻，被变爻或其它动爻所冲，依然无法让寅木入墓。待到日月冲墓之时，爻可入墓。
（4）墓爻不见合，逢冲者，爻不入墓。如寅木化未土，日冲未土，此时未土被打开，就像盒子无法关闭，自然也是关不住寅木的。如果有动爻、变爻合墓，该爻也无法墓住寅木，等日月冲墓爻可入墓。
（5）日月墓不被爻影响。日月为大，可以墓所有爻，但是日墓能产生真正作用，月墓只能取象，并不产生真正作用。例如，寅木被申金发动克制，日墓申金可以保护寅木，月墓申金不能保护寅木。
（6）伏神伏于墓爻之下，表示伏神入墓。因为飞神权限高于伏神。
（7）墓爻逢空，则失去墓其它爻的权利。等日月冲墓时，爻可入墓。

总得来说，爻入日墓就是说日是爻的墓库之地，此时爻无论是静爻还是动变爻，都论失去生克权，相当于日是一个盒子，把爻装了起来，待日月冲爻即可出墓。
爻发动，入变爻之墓代表动爻所化之变爻为动爻的墓库。例如木爻化未土。此时变爻不对动爻产生生克作用，而且把动爻包裹起来，让其失去生克权。动爻墓静爻亦是如此。

### 出墓时间

1、若爻入日月墓，冲爻则出（不可以冲日月，因为日月为大）。
2、动爻入变爻之墓，冲爻，冲墓则出。
3、爻入动爻之墓，冲爻，冲墓则出。

# 动爻
 
某根爻只要动了就会作用于其它爻，不动的爻（即静爻）不对其它爻产生作用。
 
某根爻从月、日或动爻任何一个方向得到帮助，这根爻就有气；如果一个没有，那这根爻就没有气。
 
 
 
# 爻之状态
 
## 一、静爻
 
静爻：只论有气无气；只能受作用于动爻，而不能作用于其它爻。
 
## 二、动爻与变爻
 

**动爻**化出后为**变爻**（生、克、冲、合、长生、墓绝、进、退共七种关系），变爻只能作用于动爻，不能作用于其它爻：
 
- 回头克：变爻克动爻
- 回头生：变爻生动爻
- 回头冲：变爻和动爻六冲，又称反吟
- 化成合：变爻和动爻六合
- 化长生：变爻是动爻的长生位
- 化墓绝：变爻是动爻的墓或绝地
- 化进退：变爻是动爻下一个或上一个同五行属性的地支，下一地支为进，上一地支为退
 
例：若动爻是寅木，变爻为卯木则为化进；若动爻是辰土，变爻为巳火不叫化进。 

以巳火为例：
 
- 子水：回头克
- 卯木：回头生
- 亥水：反吟、绝
- 申金：化合
- 寅木：长生
- 戌土：墓库
- 午火：化进

### 动爻之间的生克规则
动爻之间的生克规则：先克后生
动爻对静爻的生克规则：先生后克
一个卦中有动爻的，首先判断动爻之间的生克关系，再判断动爻和静爻之间的生克关系。
动爻和变爻之间，动爻不能生克冲合变爻，变爻可以生克冲本位的动爻，但不能去作用本位动爻以外的爻

动爻在本卦内的生克冲合作用有目标，都是围绕用神和世爻来运动的。
在卦中如果动爻之间组合成三合局，优先以三合局来判断动爻之间的关系。
在卦中理论上动爻可以单独作用用神和世爻，但是同性相吸，动爻和动爻之间出现连续相生，先考虑连续相生，再看最后那个动爻对用神和世爻的关系论吉凶。
在自占卦中世爻发动是不参与连续相生的，但是世爻可以参与三合局。
在代占卦或者问与自身无关的事，世爻可以参与连续相生


 
# 飞伏神

伏藏的用神为伏神（又称伏爻），伏爻的气在月看旺相休囚，在日看长生墓绝，卦中其它的爻作用不到伏爻，伏爻只受飞神（其伏藏的那根爻）作用，飞神又叫飞爻。即：伏神受月、日、飞爻作用。

 
# 月破、日破、暗动、旬空
 
**月破**：与月建六冲之爻（不管有气无气）。
**日破**：无气且与日辰相冲之爻。
**暗动**：有气且与日辰相冲之爻。
- 有气的爻被冲，不但不会冲破，还会冲动，故称暗动，暗动爻可作明动。
- 日暗动虽不会化成一根变爻，但也视为一种动爻，它的力量比动爻稍微弱些。
日破与月破的区别：月破是一种比较大的破坏，若无其它爻帮助，基本上都会没气。



 **旬空（即空亡）**，如测感情，世爻旬空，则说明卦主心里面没有底、没把握。 若用神旬空，大多不是好兆头，如追女生，用神妻财旬空，说明她心思根本不在你身上。
 


# 爻之关系——生、合、冲、克。
 
 爻和爻的关系，只有四种关系：生、合、冲、克。

 ## 冲
 
冲分为两类：
 
- 相克的冲：子午冲、寅申冲、卯酉冲、巳亥冲
- 同性的冲：辰戌冲、丑未冲
 
例：夫妻吵架，若一方子水，另一方午火（子午冲，相克之冲），则子水胜过于火，这种吵架会很快结束，不严重；但辰戌冲、丑未冲（同性之冲）测吵架会听的比较严重，因为谁也不服谁。
 
## 合
 
合的关系中，可能相克，也可能相生：
 
- 相克的合：如子丑合、卯戌合、申巳合
- 相生的合：如寅亥合、午未合、辰酉合
 
例：测夫妻感情，若为子丑合，虽然相合是夫妻，但五行相克，故为一方听从另一方；相克而合，一方要足够强才能合得上，如子丑合，子水必须够强，子水不够强则会被丑土克，合不上；同样，卯戌合，戌土也要足够强才行。
 
相生的合就比较融洽。
 
特别注意卯戌合，此为“淫合”：
 
- 若一方为男，另一方为女，当男女关系，多为非法交易；
- 若同为男性，又不存在同性恋，则多为其他不正当关系。
 
 
 
# 断卦四看
 
一看：世爻和用神
 
- 世爻为己，用神为所测之事。
- 世爻与用神都须有气。
- 用神持世、生合为兴；用神克世、冲世为败。
- 世爻或用神任何一方无气，基本上就不用继续看了。
 
 
 
二看：世爻和应爻
 
- 世为己，应为他。
- 世应比合：两情相投，比即同种五行属性。
- 世应相冲：一拍两散。
- 世克应：他随我愿。
- 应克世：我随他愿。
- 世生应：我利于他。
- 应生世：他利于我。
 
 
 
三看：动爻
 
- 动爻：分析动爻对于用神、世爻起何作用。
- 动爻生扶为吉，来冲克为凶。
 
 
 
四看：用神、元神、忌神、仇神
 
- 生用神者为元神，克用神者为忌神，生忌神者为仇神，又用神克者为仇神。
- 例：用神为官鬼（水），元神为妻财（金），忌神为子孙（土），仇神为兄弟（火）。
- 细查四神旺动，以断吉凶。
