# 2. 获取sgRNA
使用[CRISPRdirect](http://crispr.dbcls.jp/)设计sgRNA

## 2.1 sgRNA设计

<img src="/assets/images/Figure48.png" width="1000"> 

在设计出的sgRNA中选择一个最合适的，筛选标准如下；<br>
**TTTT in 20mer ：** 必须是 “-”。如果出现 TTTT，U6 启动子会提前终止转录，导致 sgRNA 无法表达。<br>
**20mer + PAM：** 必须为 1。这意味着在全基因组中它是唯一的，没有绝对脱靶风险。<br>
**12mer + PAM (Seed region) 和 8mer + PAM**  数值越小，脱靶概率越低。<br>
<img src="/assets/images/Figure49.png" width="1000"> 

最终选择的sgRNA序列(5' to 3')
```
accatcgggacgtcagagcg
```

## 2.2 sgRNA体外转录

### （1）设计体外转录模板的引物

设计引物获取sgRNA体外转录的模板，以PX458为模板，进行PCR获取体外转录的模板。<br>
**T7-SPOUT1-T289A/R292A-gRNA-F(5’ to 3’):** <br>
TTAATACGACTCACTATA<span style="color: red;">**gg**</span> <mark>accatcgggacgtcagagcg</mark>GTTTTAGAGCTAGAAATAG

合成体外转录模板的引物需要使用<span style="color: red;">**hPAGE**</span> 的纯化方式；<span style="color: red;">为提高T7 promoter的效率，在T7和sgRNA序列之间添加两个g</span>。

### （2）体外扩增获取模板

gRNA-R就从gRNA scaffold上面设计一段，大概20bp左右，用来当下游引物。

|成分|体积|
|:------:|:---:|
|T7-SPOUT1-R102-gRNA-307-F|0.6μL|
|gRNA-R|0.6μL|
|Q5® Hot Start High-Fidelity 2×Master Mix|12.5μL|
|含有gRNA scaffold的质粒模板|1μL|
|ddH2O|10.3μL|

PCR程序：   98 ℃ 15 s +（98℃ 9s + 58℃ 18s + 72 ℃ 14 s）*30 cycles + 72 ℃ 10 s

### （3）使用模板进行体外转录

使用MEGAshortscriptTM T7 Transcription Kit[T7转录说明书](https://assets.thermofisher.com/TFS-Assets/LSG/manuals/cms_055515.pdf)，按照下表配方配置反应体系，混匀后37℃孵育4-8h，加入0.5μLDNase I，37℃ 15min消化DNA，醋酸铵乙醇沉淀，纯化sgRNA，测定浓度，-80℃保存。

|成分|加量|
|:----:|:---:|
|10× Transcript Aid Reaction Buffer|2μL|
|模板DNA|120ng|
|ATP/CTP/GTP/UTP mix|	2μL|
|T7 Enzyme Mix|	2μL|
|DEPC 水|	补足20μL|


# 3. 设计ssODN作为Donor
Cas9切割位点距离点突变位置±6bp以内，可以直接使用ssODN的方法，ssODN长度切口左右各设计约40bp，总长度约90bp；<br>
如果Cas9切割位点距离点突变位置超过6bp，则需要设计5' Arm 和 3' Arm，这两个Arm的长度需要在500-1000bp左右，才能够满足同源重组的需求。<br>
由于本次设计的sgRNA使Cas9切割位置距离点突变位置±6bp以内，所以可以直接设计ssODN，序列如下：

```
hSpout1-T289AR292A donor:   ctgtgtttgctgaggcccccttccaagatgggtatgacctgaccatcgggGcAtcagagGCTggctcagatgtggcctctgcccagcttcccaacttc
```
<img src="/assets/images/Figure51.png" width="1000"> 
但是在进行ssODN合成的时候需要使用<span style="color: red;">**hPAGE**</span> 的纯化方式。

```
hSpout1-T289AR292A-F1:  GGTGTCTGCCCTGCTCAGTCCTTG  8877  69.1  336bp
hSpout1-T289AR292A-R1:  CACATCTGAGCCAGCCTCTGATGC  9212  68.0
hSpout1-T289A-WT-R1:    CATCTGAGCCGCGCTCTGACGT    9210  69.1  334bp
hSpout1-T289A-WT-F2:    GACCATCGGGACGTCAGAGCG     9179  67.2  181bp
hSpout1-T289AR292A-F2:  ACCATCGGGGCATCAGAGGCT     9180  67.5  180bp
hSpout1-T289AR292A-R2:  CGCCCAGACCTCAGTCCCTCC      9359  67.6

 94℃ 20sec  68℃ 30sec  36cycles

这6条为鉴定引物
hSpout1-T289AR292A-F1 + hSpout1-T289AR292A-R1  P的 mut
hSpout1-T289AR292A-F1 + hSpout1-T289A-WT-R1    P的 WT
hSpout1-T289A-WT-F2   + hSpout1-T289AR292A-R2  P的 WT
hSpout1-T289AR292A-F2 + hSpout1-T289AR292A-R2  P的 mut
```
合成鉴定引物使用<span style="color: red;">**DSL**</span> 的纯化方式即可。


<img src="/assets/images/Figure50.png" width="1000"> 
突变发生后sgRNA的结合位置失效，所以没有必要再额外破坏PAM序列；<br>
引物3'末端如果不能精确匹配，常常导致PCR失败，据此设计引物来进行mut和WT的PCR鉴定，只需要将突变位点置于引物3'端即可。<br>

如果是5' Arm 和 3' Arm长度需要在500-1000bp左右，则通过Q5进行PCR获取Donor，连接进T载体，并测序确认序列正确，纯化PCR产物作为Donor。<br>


# 4. 组装RNP

（1）在一个 RNase-free 离心管中混合100 pmol Cas9 和 60 pmol sgRNA，室温孵育 15-20 分钟，使RNP复合物形成（Cas9要带有核定位信号序列NLS）。<br>
（2）将 PCR 纯化后的线性1kb Donor 4μg和1μg GFP荧光表达质粒（较小的质粒）加入到上一步骤所得液体中，混合均匀。<br>

| 组分 | 建议用量 (质量比) | 备注 |
|------|-------------------|------|
| Cas9 蛋白 | 100 pmol  | Alt-R V3 活性很高，这个量足够。 |
| sgRNA | 60 pmol) | 保持与 Cas9 的质量比 4:1。 |
| 线性 Donor (1kb) | 4 μg | 大幅增加，大片段 HDR 极其依赖模板浓度。 |
| GFP荧光质粒 | 1 μg | 用于sorting细胞。 |

（3）通过电转1皿293T细胞，然后培养，通过GFP荧光进行流式sortingGFP强阳性的前20%细胞。<br>
（4）通过hSPOUT1-T289/R292-Donor-F1和hSPOUT1-T289A-screen-R PCR鉴定阳性克隆。<br>


# 5. 筛选阳性单克隆

对PCR阳性克隆进行整个 1kb 区域的全覆盖测序，确保在重组过程中没有引入新的随机突变，同时保证T289A和R292A两个点突变正确。
